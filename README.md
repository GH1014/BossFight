## BossFight

**프로젝트 개요**

BossFight는 플레이어가 보스와의 전투를 펼치는 액션 게임입니다.

방향키로 캐릭터 이동, Z키를 이용하여 슬라이딩, X키를 이용하여 공격, C키를 이용하여 점프할 수 있습니다.

게임의 목표는 플레이어가 보스와의 전투를 통해 승리하는 것입니다. 

보스는 일정 시간이 지날 때마다 4가지 다른 패턴을 이용하여 플레이어를 공격합니다.
